PHP

1.命令 wget http://hk1.php.net/get/php-5.4.45.tar.gz/from/this/mirror

2.命令 tar -zxvf mirror

3.命令 cd php-5.4.45

4.命令 ./configure --prefix=/usr/local/php5.4 --enable-fpm

5.命令 make

6.命令 sudo make install

命令

启动PHP-FPM
sudo /usr/local/php/sbin/php-fpm

查找进程任务
ps -aux |grep php-fpm

杀死进程
kill -int(quit) 进程ID
